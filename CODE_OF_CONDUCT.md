# Code of Conduct

## 1. Purpose

KProvEngine aims to maintain a professional and respectful environment for collaboration.

This Code of Conduct applies to:

- Issues
- Pull requests
- Discussions
- Project communication channels

---

## 2. Expected Behavior

Participants are expected to:

- Engage respectfully and professionally
- Provide constructive technical feedback
- Avoid personal attacks
- Focus on engineering merit and evidence

---

## 3. Unacceptable Behavior

The following is not tolerated:

- Harassment or discrimination
- Abusive language
- Personal attacks
- Deliberate disruption of technical discussion
- Disclosure of private security vulnerabilities

---

## 4. Enforcement

The repository maintainer may:

- Remove comments
- Close discussions
- Block contributors
- Revoke contribution privileges

Enforcement decisions are final.

---

## 5. Reporting

Report conduct concerns via:

security@carcodez1.dev

Reports will be handled confidentially.

---

## 6. Scope Limitations

This Code of Conduct applies only to the OSS core repository and related discussions.

Enterprise products and commercial engagements are governed by separate contractual agreements.
